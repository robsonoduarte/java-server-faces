JSF Servlet/Mapping is optional

Some application servers ( this poc was tested in JBOSS AS 7) automatically provide a servlet mapping for the /faces/*, *.faces and *.jsf patterns, provided that any of the following conditions applies;

  • Any of the classes in the web application uses a JSF annotation
  • Any initialization parameters start whit javax.faces
  • the web-inf/faces-config.xml file is present
